## Unit Testing
To run the unit tests, run the following command from the <b>root</b> directory:

```python -m pytest test\ -v``` or ```python -m pytest test\<test file>.py -v```

To hide ```DeprecationWarning```s, add:
```-W error::DeprecationWarning```
